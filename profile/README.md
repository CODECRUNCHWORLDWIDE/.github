## 🎯 What we are

**Code Crunch Worldwide** is a global, student-led open-source academy. We publish complete, project-based engineering curricula on GitHub — no paywalls, no proprietary platforms, no certificate-mill fees. Built originally at FIU, open to learners on every continent.

Every track meets the same quality bar: a SYLLABUS, weekly READMEs, lecture notes, exercises, challenges, quizzes, homework, and a project. Every track is GPL-3.0 — fork it, teach from it, translate it.

Two key documents define the academy:

- **[`MASTER-CURRICULUM.md`](https://github.com/CODE-CRUNCH-WORLDWIDE/.github/blob/main/profile/MASTER-CURRICULUM.md)** — every course, every learning path
- **[`CRUNCH-LABS-CHARTER.md`](https://github.com/CODE-CRUNCH-WORLDWIDE/.github/blob/main/profile/CRUNCH-LABS-CHARTER.md)** — the governance and quality bar for advanced tracks

## 🎓 Recommended learning paths

|Path|Sequence|Duration|
|---|---|:--:|
|**Job-ready Python developer**|C1 → C13 → C3|6–9 months|
|**Senior Python backend engineer**|C1 → C16 → C15 → C17|12–14 months|
|**Data scientist / classical ML**|C1 → C5 → C17 (perf + PyTorch)|10–12 months|
|**Cloud engineer (GCP path)**|C1 → C16 → C15 → C18|14–18 months|
|**Cloud engineer (AWS path)**|C1 → C16 → C15 → C19|14–18 months|
|**iOS engineer**|C1 → C20|9–12 months|
|**Android engineer**|C1 → C21|9–12 months|
|**AI agents engineer**|C1 → C5 → C23|12–14 months|
|**Robotics engineer**|C1 → C7 → C24|18–24 months|

Full path catalog: [`MASTER-CURRICULUM.md`](https://github.com/CODE-CRUNCH-WORLDWIDE/.github/blob/main/profile/MASTER-CURRICULUM.md).
